ข้อมูลหลักสูตร

  • ข้อมูลหลักสูตร

    • ชื่อหลักสูตร: หลักสูตรวิทยาศาสตรบัณฑิต สาขาวิชาวิศวกรรมสารสนเทศและเครือข่าย

    รหัสหลักสูตร: 25620151100293

    ระดับการศึกษา: ปริญญาตรี

    คณะ / มหาวิทยาลัย: คณะเทคโนโลยีเเละอุตสาหกรมการจัดการ / มจพ.

    คำอธิบาย: พัฒนาคน เพื่อพัฒนาวิศวกรสารสนเทศและเครือข่าย ซึ่งจะเนนการปฏิบัติเพื่อเป็นบัณฑิตที่คิดเป็น ทําเป็นที่สอดคล้องกับอัตลักษณ์ของมหาวิทยาลัยและคณะ
